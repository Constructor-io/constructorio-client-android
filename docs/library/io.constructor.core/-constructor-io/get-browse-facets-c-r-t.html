Returns a list of browse facets results. This function should be called within a coroutine scope.

Example:

runBlocking {
    launch {
        try {
            val browseFacetsResults = constructorIo.getBrowseFacetsCRT(1)
            // Do something with browseFacetsResults
        } catch (e: Exception) {
            println(e)
        }
    }
}
